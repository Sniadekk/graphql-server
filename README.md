# graphql-server

## Features
### Development
* GraphiQL
* Live Reloading of GraphQL schema

### Production
* Basic security headers from `helmet`

## Commands
### Launch webpack in development mode
```bash
npm start
```

### Run the server
```bash
npm run server
```
> The build must precede this command in order to have something to establish.


### Build
```bash
npm run build
```
